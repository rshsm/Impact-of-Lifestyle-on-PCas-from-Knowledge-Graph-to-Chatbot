# 环境说明

### 预先安装

node.js  12.8.0

npm 6.14.9 （npm会随node.js安装时默认安装）

git软件 2.28.0

koa 2.7.0

### 命令

1. 在windows的终端下先启动neo4j服务：

   `neo4j.bat console`

   启动完后打开localhost链接，在浏览器中可以看到图形化界面。

2. 在项目的根目录下右键'Git Bash Here'，然后输入下面的命令启动koa框架：

   `npm run dev`

​       然后打开localhost链接即可，注意！！！两个localhost的链接默认端口号不一样。
